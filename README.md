# hackrf

## HackRF开发环境配置

1. [Ubuntu 16.04.3 LTS(64 bit)](http://releases.ubuntu.com/16.04/ubuntu-16.04.3-desktop-amd64.iso.torrent)
	* 系统盘一定要64位的，否则U盘启动盘无法识别

2. 更新中文源

3. [pybombs](https://mirrors.tuna.tsinghua.edu.cn/help/pybombs/)一条龙可以直接装好，时间比较长

## 搭建FM收音机
